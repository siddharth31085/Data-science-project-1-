# Predictive Maintenance: Machine Failure Classification
This project uses Python and Machine Learning to predict whether a piece of industrial equipment will fail based on sensor data (temperature, rotational speed, torque, etc.).

## 🔍 Project Overview
Predictive maintenance helps industries save millions by identifying equipment issues before they lead to catastrophic failure. This project implements a **Random Forest Classifier** to detect failure patterns.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Libraries:** Pandas, Scikit-Learn, Matplotlib, Seaborn
* **Model:** Random Forest (Accuracy: ~98%)

## 📊 Key Features
- Automated data cleaning and scaling.
- Exploratory Data Analysis (EDA) of sensor correlations.
- Feature importance analysis (identifying which sensors matter most).
